Docker with Unix Domain Socket

2024-08-31
:: tags:  #docker  #unix-domain-socket

Some docker compose snippets.

unix domain socket

Bind a host Unix socket to a container port.

Refer to this.

services:
  app:
    image: someapp
  web:
    image: alpine/socat
    network_mode: service:app
    restart: always
    command:
      - unix-listen:/run/web.sock,fork,reuseaddr,mode=666
      - tcp-connect:localhost:PORT # or udp-connect:localhost:PORT
    volumes:
      - .:/run

Caddyfile Example

example.com,
*.example.com {
    tls _CERT_ _KEY_
    encode zstd gzip

    @api host api.example.com
    handle @api {
        reverse_proxy [::1]:8080
    }

    # other sites


    # fallback site
    handle {
        root * /usr/share/caddy
        file_server
    }
}
